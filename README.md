# Local Radio

A local radio station that streams music from your local filesystem.

## Features

- Stream music from your local filesystem
- Supports MP3, FLAC, and WAV files
- Supports playlists
- Supports basic metadata (title, artist, album, track number, duration)
- Supports basic authentication
- Supports basic uploading
- Supports basic streaming
- Supports basic web interface
- Supports basic web interface authentication
- Supports basic web interface uploading
- Supports basic web interface streaming

## Usage

### Docker

1. Run `docker-compose up -d` to start the radio server and rustfs
2. Open [local webpage](http://localhost:3000) in your browser
3. Upload a song to the web interface
4. Enjoy your local radio station!
